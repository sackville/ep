# The biggest discovery

Oddly enough, I don’t think it belongs in Command.

I think it belongs in Enterprise Playability.

I would create a blog post because I think we’ve discovered something genuinely interesting.

Not:

Git is useful.

Rather:

A repository can function as an organization’s collective working memory.

That’s a much richer idea.

Especially when combined with:

* Markdown
* YTHO
* MADR
* AI collaborators
* trunk-based development

I honestly don’t think that’s a Command idea.

I think that’s an Enterprise Playability idea.
